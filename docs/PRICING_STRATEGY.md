## Executive Summary

HARMONIC insight は「コンサルティング起点でツールを浸透させ、なくてはならないものにする」戦略を採る。
価格は Web 上に公開せず、パートナーとの個別協議で決定する。本ドキュメントは**社内参考価格帯**として位置付ける。

**核心**: 「ツール導入」ではなく「業務改革基盤の構築」として販売。ROI で説明可能な価格帯を設定。

## 11. 価格戦略の原則

### 11.1 ポジショニング

```
                    高 ┌─────────────────────────────────────┐
                      │                                     │
          価          │        HARMONIC insight              │
          格          │    「エンタープライズ業務 DX 基盤」    │
          帯          │                                     │
                      │   ★ MS Office 不要                   │
                      │   ★ AI ネイティブ（BYOK）            │
                      │   ★ コンサル連動                     │
                      │   ★ Windows ローカルアプリ           │
                      │                                     │
                    低 └─────────────────────────────────────┘
                      低 ──── 業務特化度 / カスタマイズ性 ──── 高
```

### 11.2 核心原則

1. **高単価 × ROI 説明**: ツールではなく業務改革の成果として価格を正当化
2. **スイート優先**: 単品よりバンドル導入を促進（40% 割引で動機付け）
3. **コンサル相乗効果**: コンサルフィー + ライセンスの一体提案で案件規模を拡大
4. **BYOK でコスト透明化**: AI 利用料はクライアント負担 → ライセンス料と分離
5. **価格非公開**: Web 上に掲載しない。「特別条件」として個別提示

### 11.3 KPI

| 指標 | 算式 |
|------|------|
| **案件単価** | コンサルフィー + ライセンス + 年間保守 |
| **年間収益** | 案件単価 × コンサル案件数 × パートナー件数 |
| **LTV** | 初期ライセンス + (年間保守 × 継続年数) |
| **TRIAL→BIZ 転換率** | 目標: 40〜60% |
