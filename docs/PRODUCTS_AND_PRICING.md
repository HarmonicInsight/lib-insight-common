## 価格方針

### 基本方針

| 項目 | 内容 |
|------|------|
| **販売形態** | 全製品 法人向け（B2B Only） |
| **販売方法** | コンサルティング案件の一環として提供 / パートナー（代理店）経由 |
| **決済方法** | Stripe（自社サイト）/ 請求書払い |
| **価格決定** | パートナー（販売代理店）との協議により個別見積もり |
| **価格公開** | **行わない**（Web サイト等での掲載禁止） |

### プラン体系（全製品共通 — 4ティア制）

| プラン | 対象 | 有効期限 | 説明 |
|:------:|------|:-------:|------|
| **FREE** | 評価・個人利用 | 無期限 | Group A: 全機能（保存/エクスポート不可）、Group B: 閲覧モードのみ |
| **TRIAL** | 評価企業 | 30日間 | 全機能利用可能（評価用） |
| **BIZ** | 法人利用 | 365日 | 法人向け全機能 |
| **ENT** | 大企業 | 要相談 | 全機能 + API/SSO/監査ログ、カスタマイズ対応 |

### 製品グループ別プラン適用

| グループ | 対象製品 | 利用可能プラン | FREE プランの挙動 |
|:--------:|---------|:-------------:|------------------|
| **Group A** | INSS, IOSH, IOSD, INPY, ISOF, INMV, INIG | FREE / TRIAL / BIZ / ENT | 全機能利用可能（保存/エクスポート不可） |
| **Group B** | INBT, INCA, IVIN | FREE / ENT のみ | 閲覧モードのみ（起動可・操作不可） |

### 価格単位

| ティア | 製品 | 単位 |
|:------:|------|:----:|
| Tier 1 | INCA, INBT, IVIN | 企業単位（per_company） |
| Tier 2 | INMV, INIG | 企業単位（per_company） |
| Tier 3 | INSS, IOSH, IOSD, INPY | ユーザー単位（per_user） |
| Tier 4 | ISOF | ユーザー単位（per_user） |

### ENT 共通機能（全製品）

| 機能 | 説明 |
|------|------|
| API利用 | 外部システムからのAPI経由アクセス |
| シングルサインオン | 企業の認証基盤との連携 |
| 監査ログ | 操作履歴の詳細記録 |
| 優先サポート | 優先的なサポート対応 |
