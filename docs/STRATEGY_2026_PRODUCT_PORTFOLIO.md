> **作成日**: 2026-02-27

## Executive Summary

HARMONIC insight は現在、5つの主力製品が市場投入可能な水準に達しつつある。これらは単なるソフトウェア群ではなく、**「コンサルティング・デリバリーの武器」**として設計された統合プロダクトスイートである。

FPT Consulting Japan のマネージングダイレクター（以下MD）としての顔と、HARMONIC insight 合同会社の経営者としての顔を**戦略的に使い分ける**ことで、通常のSaaS企業には不可能な「コンサルティング起点の製品浸透」を実現できる。

**核心戦略**: "売りに行く"のではなく、"コンサル案件の中で使わせて、なくてはならないものにする"

5製品に共通する技術的優位性：

2. **MS Office不要**: Syncfusion により Office ライセンス費用を削減（INSS/IOSH/IOSD）

4. **バージョン管理内蔵**: Excel/PowerPoint/Word に Git 相当の変更追跡を提供
5. **共通ライブラリ (insight-common)**: 全製品でライセンス・AI・ブランド・設定を統一管理
6. **日英バイリンガル**: 初日から国際展開を前提とした設計

### 2.3 利益相反の回避

```
原則: 「FPTクライアントに HARMONIC 製品を押し売りしない」

正しいフロー:
  1. FPT案件としてクライアントの課題をアセスメント
  2. 複数の選択肢を提示（市販品A、市販品B、HARMONIC製品C）
  3. クライアントが自らHARMONIC製品を選択した場合にのみ導入
  4. ライセンス契約はHARMONIC insightとして独立して締結

避けるべきフロー:
  ✗ FPT案件でHARMONIC製品ありきの提案をする
  ✗ FPTの請求にHARMONIC製品のライセンス料を混ぜる
  ✗ FPTクライアントに「MDの会社」であることを隠す
```

**透明性が最大の武器**: クライアントに「私自身が開発している製品です」と正直に伝えた上で、「だからこそ御社の要件に合わせてカスタマイズできます」とポジショニングする。隠すよりも遥かに信頼される。

### 3.1 Phase 1: 日本市場 — 即時実行（2026 Q1-Q3）

#### Wave 1: 即投入（今すぐ）— IOSH + INSS + INMV

```

ペインポイント:
  ・MS Office ライセンス費用が高い（年間数千万円規模の企業も）
  ・Excel/PowerPoint の変更管理ができない（誰がいつ変更したか追えない）
  ・社内文書の品質がバラバラ（AIによる校正・標準化ニーズ）

提案シナリオ:
  「御社のMS Officeライセンス費用を50%削減しながら、
   AIアシスタントとバージョン管理を追加できます」

価格戦略:
  MS Office 365 Business Standard: ¥1,874/user/月
  InsightOffice Suite (IOSH+INSS+IOSD): 個別見積もり
  → MS Office の 40-60% の価格帯を目安に提案
  → 初年度は導入支援込みのバンドル価格
```

**InsightCast（INMV）の攻め方:**

```
ターゲット: 人事部門 / 教育研修部門 / マーケティング部門
ペインポイント:
  ・社内研修動画の制作に外注費がかかる（1本 30-50万円）
  ・PowerPoint資料の動画化ニーズがあるが方法がない
  ・ナレーション録音のために時間と場所を確保する必要がある

提案シナリオ:
  「既存のPowerPoint研修資料を、AIナレーション付き動画に
   ワンクリックで変換。VOICEVOX の自然な日本語音声で
   外注費を90%削減」

差別化:
  ・Canva/Lumen5 → 英語圏向け、日本語TTS品質が低い
  ・InsightCast → VOICEVOX統合で日本語TTS品質が圧倒的
  ・バッチ処理で100本の研修動画を一括生成可能
```

```
提案シナリオ:
  「業務調査のヒアリングをAIが自動実行。
   30人のAIインタビュアーが24時間稼働し、
   5W1H形式で構造化された調査レポートを自動生成」

最大の武器:
  ・FPT案件で実際に使って実績を作る
  ・「FPTが自社のコンサル案件で使っているツール」という実証
  ・コンサルの"秘密兵器"を製品化したストーリー
```

### 4.2 InterviewInsight（IVIN）

**ポジショニング**: "コンサルタントの秘密兵器 — 業務調査の工数を80%削減するAI自動ヒアリングツール"

**価値提案の段階:**

```
Step 1: FPT案件で自社利用（2026 Q2-Q3）
  → 業務調査案件で InterviewInsight を実際に使う
  → ヒアリング工数削減の実績データを蓄積
  → 「FPT Consulting Japan は独自のAIツールで
     業務調査の工数を80%削減した」というストーリー

Step 2: クライアントDX部門への提案（2026 Q3-Q4）
  → 「御社の社内業務調査にもこのツールを使えます」
  → 社内コンサル部門 / 監査部門 / 品質管理部門がターゲット

Step 3: コンサルファーム・SI企業への外販（2027-）
  → 他のコンサルファームにもライセンス販売
  → 「FPTが自社で使って実証済みの業務調査AI」
```

**競合不在の独自ポジション:**

現時点で「AIが自動でビジネスヒアリングを実施し、5W1Hで構造化し、フォローアップ質問まで自動生成する」製品は**世界的に存在しない**。

- SurveyMonkey / Typeform → アンケートツール（対話型ではない）
- Otter.ai / Fireflies → 会議録音・文字起こし（ヒアリング実施はしない）
- InterviewInsight → **AIが主体的にヒアリングを実施し、構造化する**

これは「カテゴリークリエイター」になれる製品。

## 9. 競合との差別化まとめ

### 9.1 なぜ勝てるのか — 5つの構造的優位性

```
1. コンサル起点のプロダクト開発
   ─────────────────────────
   一般のSaaS企業: マーケ → リード → 営業 → 導入
   HARMONIC insight: コンサル案件 → 課題発見 → ツール提案 → 導入
   → 営業コストがほぼゼロ。既存の信頼関係の上に製品を載せる

2. FPTのASEANネットワーク
   ─────────────────────────
   スタートアップが自力で東南アジアに進出するコスト: 数億円
   FPT経由で進出するコスト: ほぼゼロ（既存拠点・既存クライアント）

3. AI ネイティブ設計
   ─────────────────────────
   MS Office: AIは後付け（Copilotは別料金¥4,497/月）
   InsightOffice: AIは標準搭載、設計段階から統合

4. データ主権（完全オンプレ可能）
   ─────────────────────────
   MS 365 / Google Workspace: データはクラウド必須
   InsightOffice: 完全ローカル動作、社内サーバーで完結
   → 金融・医療・政府機関で圧倒的に有利

5. カテゴリー不在の製品（IVIN）
   ─────────────────────────
   AIが主体的にビジネスヒアリングを実施するツールは世界に存在しない
   → カテゴリークリエイターとして市場を定義できる
```
