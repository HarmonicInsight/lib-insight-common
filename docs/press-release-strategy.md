# HARMONIC insight プレスリリース戦略

> 作成日: 2026-02-28
> 目的: 副業禁止制約下での安全なプレスリリース戦略の策定

---

## 1. 前提条件と制約

### リスク要因

| リスク | 深刻度 | 対策 |
|--------|:------:|------|
| 本業の雇用契約における副業禁止条項 | **最高** | 個人名の露出を最小化、法人格で活動 |
| プレスリリースによる本業への露出 | **高** | 配信先・媒体の選定で制御 |
| SNS 等での拡散リスク | **中** | 個人 SNS との紐付けを避ける |
| 競業避止義務への抵触 | **要確認** | コンサル業務と製品事業の領域分離を明確化 |

### 法的留意事項（先に対応すべき）

1. **雇用契約書の確認**: 副業禁止の範囲（完全禁止 vs 届出制 vs 競業のみ禁止）
2. **合同会社の代表社員**: 登記上の代表は公開情報 → 法人番号から辿れる点を認識
3. **弁護士・社労士への相談**: プレスリリース前に、雇用契約上のリスクを専門家に確認することを**強く推奨**

## 2. プレスリリース戦略：3段階アプローチ

### Phase A: 製品・パートナー主導型（推奨 — 今すぐ実行可能）

**方針**: 個人名を出さず、法人名と製品にフォーカス

```
露出レベル: ★☆☆☆☆（最小）
個人名出現: なし
リスク: 低
効果: パートナー獲得に十分
```

| 項目 | 内容 |
|------|------|
| **発信主体** | HARMONIC insight合同会社（法人名のみ） |
| **引用コメント** | 「HARMONIC insight合同会社 開発チーム」等、個人名なし |
| **連絡先** | 法人メール（press@harmonicinsight.com 等）、電話番号は法人用 |
| **配信先** | PR TIMES / @Press（B2B IT 系メディア向け） |
| **写真・動画** | 製品スクリーンショットのみ。人物写真は使用しない |

### Phase B: 代表者限定露出型（本業退職検討時）

**方針**: 代表社員として最小限の個人名露出

```
露出レベル: ★★★☆☆（中）
個人名出現: 代表社員コメントとして1箇所のみ
リスク: 中
効果: 信頼性向上
```

### Phase C: フル露出型（独立後）

**方針**: 創業者ストーリーを含むフルスケールPR

```
露出レベル: ★★★★★（最大）
個人名出現: 全面
リスク: なし（独立済み）
効果: 最大
```

## 5. 副業禁止対策：プレスリリースでの個人名回避テクニック

### やるべきこと

| 項目 | 具体策 |
|------|--------|
| **発信元** | 「HARMONIC insight合同会社」のみ。個人名を記載しない |
| **引用コメント** | 「当社開発チーム」「当社」で統一。「代表」のコメントは入れない |
| **連絡先** | 法人メール・法人電話のみ。個人携帯は不可 |
| **写真** | 製品スクリーンショットのみ。人物写真は使わない |
| **SNS 連携** | 法人アカウント（@harmonic_insight 等）のみ。個人 SNS から共有しない |
| **メディア取材** | Phase A では原則お断り。書面回答のみ対応 |
| **登壇・セミナー** | Phase A では自社主催は避ける。パートナー主催イベントでの製品紹介に限定 |

### やってはいけないこと

| NG 行動 | リスク |
|---------|--------|
| プレスリリースに個人名を記載 | 本業に発覚するリスク最大 |
| 個人 LinkedIn で共有 | FPC 関係者に露出 |
| テック系イベントで登壇 | 業界内で認知される |
| テレビ・動画メディアへの出演 | 物理的な特定 |
| 「元○○」「現職○○」等の経歴記載 | 直接的な紐付け |

## 7. プレスリリース配信前チェックリスト

- [ ] 弁護士・社労士に雇用契約上のリスクを確認済み

- [ ] 価格情報が記載されて**いない**（個別見積もり方針に準拠）
- [ ] 法人メールアドレス（press@ / partner@）が受信可能な状態
- [ ] 法人 Web サイトにパートナープログラムの情報ページが存在する

## 10. 長期的な出口戦略との関係

プレスリリースは段階的に露出レベルを上げる設計です。

```
現在（Phase A）              独立準備期（Phase B）        独立後（Phase C）
┌──────────────┐           ┌──────────────┐           ┌──────────────┐
│ 法人名のみ    │    →      │ 代表として   │    →      │ 創業者       │
│ 個人名なし    │           │ 最小限露出   │           │ ストーリー   │
│ 製品フォーカス │           │ 業界イベント │           │ メディア露出 │
│ パートナー募集 │           │ 限定登壇     │           │ 積極的PR     │
└──────────────┘           └──────────────┘           └──────────────┘
```

本業を退職し独立するタイミングで Phase C に移行することで、
創業者ストーリーを最大限活用したPRが可能になります。
それまでは Phase A で製品とパートナーネットワークの実績を積み上げ、
独立時に十分な事業基盤がある状態を目指します。
