# HARMONIC insight - Google Play デベロッパーページ ヒーロー画像プロンプト

## 概要

| 項目 | 値 |
|------|-----|
| デベロッパー名 | HARMONIC insight |
| 用途 | Google Play デベロッパーページ ヒーロー画像 |
| サイズ | 4096×2304 px |
| アスペクト比 | 16:9 |

### コンセプト

| 項目 | 値 |
|------|-----|
| テーマ | Professional Intelligence |
| キーワード | 洗練 / 信頼 / 知性 / 効率 / 先進性 |
| パーソナリティ | プロフェッショナルに寄り添う、信頼できるパートナー |

> 特定のアプリを掲載しない。コンセプトを抽象的に表現し、
> アプリが増減しても差し替え不要なデザインとする。

---

## デザイン方針

- **背景**: アイボリー（#FAF8F5）〜 ウォームゴールド（#D4A84B）のグラデーション
- **プライマリ**: ゴールド（#B8942F）— ロゴ、アクセント、装飾
- **テキスト**: ダークブラウン（#1C1917）、サブテキスト（#57534E）
- **フォント**: Inter / Noto Sans JP、太字のキャッチコピー
- **トーン**: 洗練・信頼・知性。法人品質のプロフェッショナルな印象
- **表現**: 個別アプリではなく「日常を支えるテクノロジー」の抽象表現

## 使い方

1. 上記プロンプトを画像生成 AI（Gemini Imagen、DALL-E、Midjourney 等）に入力
2. 生成結果を確認し、全体のトーン・構図が意図通りか確認
3. テキスト部分（ロゴ・タグライン）は AI 生成だと文字化けしやすいため、Figma / Canva 等で差し替え推奨
4. 4096×2304 px で出力し、Google Play Console のデベロッパーページにアップロード

## 注意事項

- 特定のアプリ画面・端末モックは含めない（アプリ追加時に画像の差し替えが不要）
- ロゴテキスト「HARMONIC insight」は後処理で正確に載せ直すことを推奨
- 青色（#2563EB）をプライマリカラーとして使用しないこと（ブランドガイドライン厳守）
- カラーコードは `brand/colors.json` の定義に準拠
