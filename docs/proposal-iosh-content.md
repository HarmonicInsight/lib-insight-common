## ページ 3: 御社の課題（共感）

### こんな経験、ありませんか？

#### 課題 1: 「最新版はどれ？」問題
```
売上報告_最終.xlsx
売上報告_最終2.xlsx
売上報告_最終_修正.xlsx
売上報告_最終_修正_田中確認済.xlsx
売上報告_FINAL.xlsx       ← どれが本物？
```

> 調査結果: 日本のオフィスワーカーの **78%** が
> Excelファイルのバージョン管理に困った経験あり

#### 課題 2: 「誰がこのセルを変えた？」
- 予算表の数字がいつの間にか変わっていた
- 変更した人が分からず、会議で確認するのに30分消費
- 変更履歴がないため、元に戻せない

#### 課題 3: Microsoft Office のコスト負担
- Microsoft 365 E3: 月額 4,497円/ユーザー
- 10人で年間 **539,640円**
- 機能の 80% は使っていない

#### 課題 4: 定型作業の属人化
- 毎月の集計作業を特定の人しかできない
- マクロが複雑化して誰もメンテナンスできない
- 退職・異動で業務が停止するリスク

## ページ 6: AI アシスタントのデモ

### AI が Excel を直接操作します

**操作例 1: 自然言語で指示**
```
あなた: 「A列の売上合計をB10に入れて」
AI:     B10 に =SUM(A1:A9) を入力しました ✓
```

**操作例 2: データ分析**
```
あなた: 「この売上データの傾向を教えて」
AI:     Q1→Q2で15%増加、Q3で横ばい、Q4で急増。
        季節要因として年末商戦が影響していると推測されます。
```

**操作例 3: 数式の説明**
```
あなた: 「C5のこの複雑な数式を説明して」
AI:     この VLOOKUP+IF の組み合わせは、
        マスタテーブルから商品名を検索し、
        在庫が0の場合は「欠品」と表示する処理です。
```

### 6つのAIツール

| ツール | 機能 |
|--------|------|
| セル操作 | 値の入力・数式の挿入・書式設定 |
| データ分析 | 傾向分析・異常値検出・要約 |
| 数式支援 | 数式の生成・解説・デバッグ |
| レポート生成 | データからのレポート自動作成 |
| 品質チェック | データの整合性・重複・欠損チェック |
| Python実行 | 高度なデータ処理のPythonスクリプト実行（PRO+） |

## 補足: デモ実施時のポイント

### デモで見せるべき「衝撃の瞬間」3つ

1. **AIがセルを直接操作する瞬間**
   - 「A列の合計をB10に入れて」→ AIが即座に実行
   - 顧客の反応: 「え、AIがExcelを操作するの？」

2. **差分比較で変更箇所が色分けされる瞬間**
   - 2つのバージョンを比較 → セル単位で赤・緑・黄に色分け
   - 顧客の反応: 「これ欲しかった！」

3. **MS Office なしで xlsx を新規作成する瞬間**
   - 「新しいスプレッドシート」→ 空のxlsxが生成される
   - 顧客の反応: 「Officeなしで作れるの？」

### デモ用サンプルデータ

| データ | 用途 |
|--------|------|
| 売上報告.xlsx（3バージョン） | バージョン管理・差分比較のデモ |
| 予算管理.xlsx（変更ログ付き） | セル変更ログのデモ |
| 空のスプレッドシート | 新規作成 + AI操作のデモ |
