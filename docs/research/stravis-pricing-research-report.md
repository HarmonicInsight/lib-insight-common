# Stravis / StravisLINK ライセンス価格の調査・推定と会計データ収集ツール比較レポート

> 調査日: 2026年2月22日
> 為替基準: 1USD=157円（日本銀行 2026/2/20公表、令和8年3月中適用）

## エグゼクティブサマリー

Stravis と StravisLINK（当時表記：STRAVIS / STRAVIS-LINK）の提供価格として、**「STRAVIS：1ライセンス600万円から」「STRAVIS-LINK：1ライセンス400万円から（税別）」**が、プレスリリース本文（2013年6月17日付）に明記されている。

現在の製品サイトや販売・導入パートナー情報では、**最新の価格表（円建て・税区分つき）を公開していないケースが多く、見積取得が前提**になっている。

StravisLINK XB（Web収集ツール）は、製品サイト上で利用シーンや位置づけ（例：BEPS対応、Excel/メール中心の収集からの置き換え等）が語られるものの、**ライセンス単価の公開は確認できなかった**。

推定では、公開価格がある類似製品（DataSpider、ASTERIA Warp、CData系）を年額換算して比較し、中堅企業（連結対象10〜50社）想定で **StravisLINK XB は「年額80万〜250万円程度（サブスク換算）」または「買い切り300万〜600万円程度＋保守（要確認）」**のレンジを暫定提示する。

## 調査スコープと前提

- Stravis / StravisLINK について、**公式サイト・一次資料（プレスリリース等）・国内販売導入パートナー情報**から、ライセンス形態／価格（円、税別/税込）／導入費用・保守費用の有無を確認して要約（出典URLを明示）
- StravisLINK XB の**公開単価が確認できない**前提で、代理店資料の記載有無を確認し、見つからない場合は **DataSpider / ASTERIA / CData など公開価格のある類似領域から推定レンジ**を提示（中堅企業：連結対象10〜50社）
- 会計データ収集に使われるツール（ETL / iPaaS / RPA / データコネクタ / Excel連携・フォーム等）を国内外から列挙し、比較表を作成（必須：Forguncy）
- 為替換算が必要な箇所は、**最新の日本銀行公表レート**として、2026年2月20日公表の「基準外国為替相場」（令和8年3月中適用：1USD=157円）を使用
- 「買い切り（永続）ライセンス」を年額換算する場合は、税務上の耐用年数として国税庁が示す「ソフトウエア：その他のもの5年」を根拠に、**（買い切り価格 ÷ 5年）**を基本とする（ただし保守費用が別途必要な製品は加算、未公開は除外）

## InsightSheetへの示唆（本レポートの戦略的活用）

### 価格ポジショニングの裏付け

| 項目 | STRAVIS-LINK | InsightSheet | 差分 |
|------|-------------|-------------------|------|
| 初期ライセンス | 400万円〜（税別） | **200万円**（税別） | **50%以下** |
| 年間保守 | 推定200万円〜/年 | **30万円/年**（15%） | **85%以下** |
| 導入支援 | 500万〜2,000万円 | **50〜150万円** | **90%以下** |
| 初年度合計 | 1,500万〜3,000万円 | **250万〜380万円** | **1/10以下** |

### 競合との差別化ポイント

1. **StravisLINK XB（直接競合）**: 価格非公開だが推定80〜250万円/年 → InsightSheetの200万円買い切りは**TCOで圧倒的優位**
2. **Forguncy（間接競合）**: 基本54万円と安いが、**会計データ収集のドメイン知識がない**。テンプレート・バリデーション・ワークフローを自分で構築する必要あり
3. **kintone（間接競合）**: ユーザー課金で少人数なら安いが、**50拠点×数名のスケールではInsightSheetの方が安い**
4. **ETL/iPaaS系**: 技術者向けで**現場の入力UIがない**。InsightSheetはExcelで入力→自動抽出のUXが差別化

### セールストーク用の比較ポイント

- 「STRAVIS-LINK導入総額の**1/10以下**で、汎用データ収集ができます」
- 「Forguncyと違い、**会計データ収集に特化したテンプレートとバリデーション**が標準搭載」
- 「kintoneのように**ユーザー数で課金されない**ので、拠点が増えてもコストが変わりません」
- 「ETLツールと違い、**現場はExcelで入力するだけ**。IT部門の負荷ゼロ」

## 未確定情報と推奨アクション

### 未確定情報

- STRAVIS/StravisLINKの**現行版の最新価格表・課金単位・年間保守費率**は非公開
- StravisLINK XBの**ライセンス単価や課金単位**は非公開
- **導入支援費用の具体的レンジ**は「別途見積」のみ

### 推奨アクション

1. **電通総研へ見積依頼**: Stravis＋StravisLINK XBの現行価格を確認
2. **パートナー経由の見積**: ビジネスブレイン太田昭和経由でも概算取得
3. **相見積の比較軸**: DataSpider / ASTERIA / CData でも同等要件で見積取得し、推定レンジの妥当性を検証

### 見積依頼時の必須伝達項目（中堅企業10〜50社想定テンプレ）

- 連結対象社数（10〜50社のどのレンジか、海外有無、通貨有無）
- 子会社側の入力者数・ピーク同時利用（概算で可）
- 収集対象（FS、内部取引、CF、注記、非財務/サステナ等）と頻度（月次/四半期/年次）
- 既存会計/ERP→収集方法（CSV中心か、DB/API連携希望か、RPA許容か）
- 保守に含まれる範囲（問い合わせ、バージョンアップ頻度、STRAVIS PASの要否）
- 導入支援の範囲（収集シート整備、チェックルール設計、教育、決算立上げ支援）
