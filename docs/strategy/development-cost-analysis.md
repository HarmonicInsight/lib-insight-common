# 競合開発コスト分析 & 競合価格試算
## ── 他社が同等ツールを開発・販売する場合のフルコスト構造と損益分岐点 ──

> 作成日: 2026年3月1日
> 目的: 競合が同等製品を市場投入した場合の現実的な価格帯（競合価格）を導出し、
>        HARMONIC insight の価格競争力を定量的に裏付ける
> 想定読者: 経営戦略、営業資料の裏付けデータ

---

## 1. 分析の枠組み

### 問い

> 「もし競合他社が HARMONIC insight と同等のツールをゼロから開発し、
>    製品として販売するとしたら、最低いくらで売らなければならないか？」

### 試算の流れ

```
Step 1: 開発チーム運営の全コストを積み上げ（人件費だけでなく全経費）
Step 2: 初期開発完了後の年間運営コスト（保守・サポート・インフラ）
Step 3: 販売に必要な営業・マーケティング・管理費を加算
Step 4: 目標利益率を織り込み、必要売上を逆算
Step 5: 顧客数シナリオ別に「損益分岐の製品単価」= 競合価格 を導出
```

## 3. 製品別 開発コスト（フルコスト）

### コスト計算式

```
実プロジェクト工数 = 純粋開発人月 × 1.6（オーバーヘッド乗数）
人件費+環境費     = 実プロジェクト工数 × ¥1,100,000（フル・ローデッド加重平均）
サードパーティ費   = 開発期間中に必要な外部ライセンス・サービス
────────────────────────────────────────────────
初期開発コスト合計  = 人件費+環境費 + サードパーティ費
```

## 5. 損益分岐分析 & 競合価格の導出

> **注意: このセクションの「競合価格」は、競合が損益分岐するために付けなければならない最低価格。**
> **顧客が ROI を考慮して「出せる」価格（市場価格）はセクション 6 で別途算出する。**

### 5-1. 計算モデル

```
必要年間売上 = (初期開発コスト ÷ 投資回収年数) + 年間運営コスト + 固定費按分
             ─────────────────────────────────────────────────────────
                     1 − 営業コスト率 − 目標利益率

競合価格（年額/ライセンス） = 必要年間売上 ÷ 顧客数
```

**前提パラメータ:**

| パラメータ | 値 | 根拠 |
|-----------|---:|------|
| 投資回収年数 | 5年 | SaaS/ソフトウェア業界の標準 |
| 営業コスト率（S&M + G&A） | 35% | B2B SaaS の中央値（Bessemer BVP） |
| 目標営業利益率 | 15% | 成長期 B2B ソフトウェア企業の目標水準 |
| 固定費按分 | 製品数で均等割り | 10製品で按分 |

## 7. 市場価格（顧客ROIベース）

> **市場価格 = 顧客が ROI を考慮して「出せる」と判断する価格帯**
>
> 算出方法: 製品が削減する業務コスト（年間価値）の **5〜10%** を市場価格とする。
> 日本の B2B 市場では「得られる価値の 10%」が購買判断の心理的閾値（target-market-swot-pricing-2026.md）。
> コンサルタント時給 ¥30,000（シニアクラス中央値）で算出。

### 7-1. 顧客の支出コンテキスト — ツールは「コンサル費の一部」

HARMONIC insight の製品は **単体のソフトウェア** として購入されるのではなく、
**コンサルティング案件（数百万〜数千万円）の中で導入される業務改善ツール** として位置付けられる。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  顧客の典型的な年間 IT / コンサル支出

  コンサルティングフィー:     ¥5,000,000〜¥50,000,000/年
  ├ 業務分析・ヒアリング:     ¥2,000,000〜¥10,000,000
  ├ 提案書・報告書作成:       ¥1,000,000〜¥5,000,000
  ├ RPA 導入・移行:          ¥5,000,000〜¥30,000,000
  └ 運用保守:                ¥1,200,000〜¥4,200,000/年

  ─────────────────────────────────────────────────
  その中での HARMONIC insight ツール:
    BIZ ライセンス:           ¥52,800〜¥217,800/年/PC
    → コンサル費の 0.1〜4% にすぎない
    → 稟議で「ツール代」として問題にならない金額
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**顧客の購買心理:**

| 状況 | 顧客の反応 |
|------|-----------|
| コンサル案件 ¥1,000万の中で「このツール ¥10万/年です」 | 「コンサル費の 1%。誤差。導入してください」 |
| コンサル案件 ¥3,000万の中で「このツール ¥20万/年です」 | 「安い。何台でも入れてください」 |
| ツール単体で「¥20万/年です」（コンサル文脈なし） | 「他のツールと比較したい」← **この状況を避ける** |

→ **HARMONIC insight はコンサル案件の一環として提案される限り、価格は「誤差」として認知される。
  ツール単体販売ではなく、コンサルティング価値の延長として位置付けることが重要。**

### 8-5. プレミアムブランド戦略 — 「良いもの＝高い」

> **コスト構造の優位性 ≠ 安売り。Prada と同じ。**
>
> Prada はイタリアの自社工場で高い品質管理のもと製品を作るが、
> 製造原価が低いからといって安売りはしない。
> ブランドの価値で価格を決め、結果として高い利益率を得る。
>
> HARMONIC insight も同じ構造を持つ:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Prada モデルの適用

  ┌─────────────────────────────────────────────────────────┐
  │                                                         │
  │  「原価が安い」→「安く売る」  ← これは間違い（ユニクロ）  │
  │                                                         │
  │  「原価が安い」→「ブランド価値で価格を決める」            │
  │               →「結果として圧倒的利益率」                │
  │               →「利益を品質・サポート・開発に再投資」      │
  │               → これが正解（Prada / HARMONIC insight）    │
  │                                                         │
  └─────────────────────────────────────────────────────────┘
```

**HARMONIC insight の価格設定原則:**

| 原則 | 内容 |
|------|------|
| **良いもの＝高い** | 価格そのものがブランドのシグナル。安くすれば「安いもの」と認知される。高い価格は品質と専門性の証明 |
| **価格はブランド価値で決める** | 「コンサルタントが業務で使う専門ツール」としての位置付け。価格は開発コストではなく「顧客が得る価値」から設定 |
| **安売りは一切しない** | セール、クーポン、期間限定割引は実施しない。価格交渉にも応じない |
| **コスト優位性 = 利益率** | 競合が 6〜18 倍の価格でなければ採算が合わない構造 → HARMONIC insight は同じ価格でも圧倒的利益率を確保 |
| **利益は再投資に回す** | 高い利益率 → 機能追加・品質向上・AI モデル最新化に継続投資 → ブランド価値のさらなる向上 |
| **希少性を維持する** | 大量販売・大量割引ではなく、コンサル案件を通じた高付加価値提案 |

> **ヴェブレン効果（Veblen Effect）**: 価格が高いほど需要が増す現象。
> 法人向け専門ツールでは「安い＝品質が低い」「高い＝信頼できる」と認知される。
> 年間 ¥10万〜¥20万の価格帯は「個人で判断できない → 稟議を通す → 組織としての意思決定」
> となり、製品の重要性が社内で認知される。これが長期的なリテンションにつながる。

**この構造が意味すること:**

| 指標 | 競合（36名体制） | HARMONIC insight |
|------|:--------------:|:---------------:|
| 100社時の損益分岐価格 | ¥70万〜¥140万/年 | **はるかに下** |
| 実際の販売価格 | ¥70万〜¥140万/年（ギリギリ黒字） | **¥5万〜¥22万/年（高利益率）** |
| 営業利益率 | 0〜5%（薄利） | **極めて高い** |
| 価格決定の自由度 | なし（コスト制約） | **完全に自由（ブランドで決定）** |
| 値下げ圧力への耐性 | 脆弱（コスト割れリスク） | **極めて強い** |

→ **HARMONIC insight は「安いから売れる」のではなく、「コンサルグレードの専門ツール」として売る。
  価格は顧客が得る価値（年間¥1,800万の業務効率化）の 1% 未満であり、
  その結果として競合が構造的に追随できない利益率を実現する。**
