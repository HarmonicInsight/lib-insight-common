## Phase 1: 集客第一主義の確立（第1〜2章）

### 1-1. ビジネスの4ステップ現状診断（第1章）

```
集客 → 販売 → サービス提供 → リピート化

現状:
  集客:          × 仕組みなし。コンサル先への口頭紹介のみ
  販売:          △ FREE→TRIAL→BIZ/ENT のプラン設計はある。実売ゼロ
  サービス提供:   ○ 製品は完成済み。5製品とも動く
  リピート化:     △ SaaS月額/年額の自動更新はある。CSプログラムなし

診断結果:
  ボトルネックは「集客」。圧倒的に。
  サービス提供（製品開発）に99%の時間を使ってきた。
  これを「集客70% : 開発30%」に逆転させる。
```

### 1-2. 2ステップマーケティングの設計（第2章）

```
小山理論: 購買プロセスは「教育 → 販売」の2段階。
         いきなり売るのではなく、まず悩みを自覚させる。

致命的な問題:
  現在の HARMONIC insight には「教育」がない。
  FREE版をダウンロードした人も「何に使うか」がわからない。
  コンサルの口頭説明は教育の代替だが、スケールしない。
```

#### 5製品の「教育」設計

**小山理論の教育の定義**: 「悩みを自覚していない人に、悩みを自覚させ、いますぐ行動させる」

| 製品 | 悩みを自覚していない人 | 自覚させるメッセージ | 行動させる仕掛け |
|------|----------------------|---------------------|-----------------|
| **Cast** | 「動画制作は外注するもの」と思っている広報担当 | 「外注1本¥300,000。Insight Training Studioなら1本¥596。自分で作れます」 | Insight Training Studioで作った動画自体が証拠。見せるだけで教育完了 |
| **Slide** | 「PPTのレビューは手動でやるしかない」と思っているコンサルMgr | 「150枚のPPTレビュー、手動2日。AIなら30分」 | Before/After動画（手動vsAI）をCastで制作 |
| **Sheet** | 「Excelのバージョン管理なんてものがある」と知らない人 | 「report_v2_final_FINAL.xlsx、あなたのPCにもありませんか？」 | FREEでバージョン管理を体験 → 「AI機能も欲しい」 |
| **Doc** | ドキュメント管理の問題を「仕方ない」と諦めている人 | 「提案書3本の整合性チェック、手動で何時間かけてますか？」 | Suite バンドルの一部として。Doc単独での教育は行わない |
| **Interview** | 「ヒアリングは会議室で1対1でやるもの」と思っている人事 | 「100人の本音、1週間で全部聞けます。会議室の予約は不要です」 | デモ動画（VRMアバターが実際にインタビューする様子）が最強の教育 |
