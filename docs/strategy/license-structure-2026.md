## 基本方針

```
1. B2B専業（個人向け販売は行わない）
2. 全デスクトップ製品は PC端末ライセンス（1ライセンス = 1PC）
3. InterviewInsight（SaaS）のみユーザーベースライセンス
4. AIクエリはプランに内包（チケット別売りではない）
5. TRIAL は1か月間・全機能無制限（自社が発行）
6. ライセンス未取得でも使える FREE モードを全製品に用意
```

## プラン共通仕様

| プラン | 説明 | ライセンス | 有効期限 | AI利用 |
|--------|------|-----------|----------|--------|
| **FREE** | 基本機能のみ。ライセンス不要で利用可能 | 不要 | 無期限 | — |
| **TRIAL** | 全機能無制限（評価用）。自社が発行 | **自社発行** | **1か月間** | Premium含む全機能無制限 |
| **BIZ** | 法人1端末・個人ユース。全機能 | 購入 | 365日 | 月200回（Standardティア） |
| **ENT** | 法人複数端末・コラボレーション。全機能＋管理機能 | 契約 | **個別見積もり** | 無制限（Premiumティア含む） |

### プラン構造の設計思想

```
FREE（ライセンス不要・基本機能のみ）
  ↓ 「これ良さそうですね」→ 営業/Webから TRIAL を発行
TRIAL（1か月・全機能無制限・自社が発行）
  ↓ 評価完了 → 稟議
  ├─ BIZ（法人1端末 = 個人ユース）
  │    1名が1台のPCで全機能を利用。
  │    コラボレーション機能なし。
  └─ ENT（法人複数端末 = コラボレーション）
       複数名が複数PCで利用。
       共同編集・管理・監査機能あり。
```

### BIZ と ENT の根本的な違い

```
BIZ = 1人で使う（個人ユース）
  → 全機能が使える。ただし「自分だけ」。
  → ライセンスは1PC単位で購入。

ENT = チームで使う（コラボレーション）
  → BIZ の全機能 + 複数人での共同作業機能。
  → 掲示板、変更者表示、VRMアバター会議、監査ログ、SSO等。
  → ライセンスは複数PC一括契約。
```

## FREEモードの設計

### なぜ FREE が必要か

```
TRIALライセンスは自社が発行する
  → ユーザーが自力でTRIALを取得する手段がない
  → FREEモードがなければ「まず触る」ことすらできない

FREE があることで:
  1. ダウンロード → すぐ使える（営業への問い合わせ不要）
  2. 基本機能で価値を体感
  3. 「全機能使いたい」→ 営業に TRIAL を依頼
  4. TRIAL で全機能評価 → 稟議 → BIZ/ENT 購入

B2B の稟議プロセスにも対応:
  - TRIAL終了後、予算承認を待つ間もFREEで業務継続可能
  - 完全ロックすると再インストールの手間でユーザーが離脱する
```

## AIクエリ内包モデル

### 旧設計（チケット制）→ 新設計（プラン内包）

| | 旧設計 | 新設計 |
|---|---|---|
| 方式 | ¥50/query 別売り | **プランに月間上限で内包** |
| ユーザー体験 | 「使うたびにお金が減る」 | 「月〇〇回まで自由に使える」 |
| 心理的効果 | 損失回避 → AI使わない | 安心感 → AI使い倒す |
| 予算 | 予測不能 | 月額固定（80%のユーザー） |
| 超過時 | — | 顧客が自身でAPI契約 |

### 設計基準

- **80%のユーザーが月間上限に達しない** 設計
- 超過するヘビーユーザーは自身でAnthropicのAPIキーを購入して利用
- APIコストは年50倍で下落中 → マージンは時間とともに改善
- 将来的なコスト下落分は「月間上限の引き上げ」でユーザー還元

## ライセンスキー形式

```
{製品コード}-{プラン}-{YYMM}-{HASH}-{SIG1}-{SIG2}

プランコード:
  FREE → ライセンスキー不要（キー未入力 = FREE）
  TRL  → TRIAL
  BIZ  → Business
  ENT  → Enterprise

例:
  IOSH-BIZ-2603-A1B2-C3D4-E5F6    ← InsightSheet Business 2026年3月
  INSS-ENT-2607-F7G8-H9I0-J1K2    ← InsightSlide Enterprise 2026年7月
  INMV-BIZ-2604-L3M4-N5O6-P7Q8    ← InsightCast Business 2026年4月
  ISOF-ENT-2605-R9S0-T1U2-V3W4    ← InsightSeniorOffice Enterprise 2026年5月

TRIAL:
  IOSH-TRL-2603-X5Y6-Z7A8-B9C0    ← InsightSheet Trial 2026年3月
```

## 実装ノート（cross-lib-insight-common）

```
ライセンス判定フロー:

  1. ライセンスキーの有無を確認
     - キーなし → FREE モード
     - キーあり → 以下のフローへ
  2. ライセンスキーの検証（形式 + 署名）
  3. 製品コードの一致確認
  4. プラン判定 → 機能フラグ/AI上限の適用
     - TRL → 全機能有効、AI無制限（Premiumティア）
     - BIZ → 全機能有効、AI月200回（Standardティア）
     - ENT → 全機能有効、AI無制限（Premiumティア）、コラボレーション有効
  5. 有効期限チェック
     - TRIAL: 初回アクティベーションから1か月
     - BIZ: アクティベーションから365日（年額更新）
     - ENT: 契約期間に基づく
  6. AI月間クエリカウンターのリセット（毎月1日）
  7. 有効期限切れ → FREE モードへの自動降格
```
