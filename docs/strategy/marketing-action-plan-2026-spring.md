## 1. 戦略の前提

| 要因 | 内容 |

## 12. パートナーリセラープログラム（書籍第8章: 仕組み化 + 第10章: 紹介）

### 12-1. コンセプト

個人コンサルタントが**自分の顧客に、自分の名前で**HARMONIC insight製品を販売できるリセラーモデル。
「紹介報酬」ではなく「自分のサービスラインの拡張」として機能する。

```
┌───────────────────────────────────────────────────────────────┐
│                   パートナーリセラーモデル                       │
│                                                               │
│  HARMONIC insight（卸値: 定価の70%）                           │
│          │                                                    │
│          ↓                                                    │
│  パートナー（個人コンサル）                                     │
│  ── 自分の名前・自分の価格で顧客に販売                         │
│  ── 導入支援・業務フィットのアドバイスも自分で提供              │
│          │                                                    │
│          ↓                                                    │
│  パートナーの顧客（エンド企業）                                 │
│  ── コンサル契約の一部としてツールを導入                       │
│  ── ツール単独の稟議が不要（コンサル契約に含まれる）           │
│                                                               │
│  【ポイント】                                                  │
│  ・1階層のみ（MLMではない）                                    │
│  ・パートナーは「紹介者」ではなく「販売者」                    │
│  ・自分の付加価値を上乗せして販売できる                        │
│  ・コンサルにとっては副収入ではなく「提案力の強化」            │
└───────────────────────────────────────────────────────────────┘
```

### 12-2. なぜこのモデルが日本市場で機能するか

| 理由 | 説明 | 書籍の原則 |
|------|------|-----------|
| **コンサルは「道具」を持ちたい** | 「うちはこのツールで分析します」と言えるだけで提案の説得力が上がる | 第5章: 差別化 |
| **既存顧客への追加提案** | 新規開拓ではなく信頼関係のある顧客への提案。ハードルが極めて低い | 第6章: 信頼 |
| **MLM臭がゼロ** | 自分の名前で売り、自分が責任を持つ。純粋なBtoB代理店モデル | ─ |
| **コンサル契約に内包できる** | 月額5万のツールを「分析環境構築込み月10万」で提供。ツール単独の稟議が不要 | 第4章: 問題解決を売る |
| **個人コンサルの数だけチャネルが増える** | 代表が営業しなくても販売チャネルが拡大する | 第8章: 仕組み化 |

### 12-3. 収益モデル例

```
例: InterviewInsight（月額5万円・定価）

  HARMONIC insight → パートナーへの卸値: 3.5万円（定価の70%）
  パートナー → 顧客への販売価格: 8万円（ヒアリング設計支援込み）

  パートナーの利益: 4.5万円/月
  HARMONIC insightの収益: 3.5万円/月（直販5万より低いが、営業コストゼロ）

  パートナーが5社に導入 → パートナー月収22.5万円 / HARMONIC insight月収17.5万円
```

### 12-4. 実行ロードマップ

| フェーズ | 時期 | やること | 完了基準 |
|---------|------|---------|---------|
| **Phase 0: 非公式スタート** | 2月〜3月 | 社内メンバーで既に個人顧客を持っている2〜3人に非公式に声をかける。契約書なし、口頭ベースで開始 | 最初のパートナー候補2〜3人が合意 |
| **Phase 1: 実績作り** | 3月〜4月 | パートナー経由で最初の1〜2件の成約を目指す。課題・成功パターンを記録 | パートナー経由の成約1件以上 |
| **Phase 2: 制度設計** | 4月〜5月 | Phase 1の実績をもとにパートナー契約テンプレート作成。卸値・責任範囲・ブランド使用ルールを正式化 | 契約テンプレート完成 |
| **Phase 3: パートナーキット整備** | 5月〜6月 | 顧客向け説明資料・デモ手順・FAQ集をパートナーに提供。「この資料を渡すだけ」レベルに簡素化 | キット一式完成 |
| **Phase 4: 拡大** | 6月〜 | 外部のフリーランスコンサル・独立系コンサルへの展開。パートナー募集ページをサイトに設置 | パートナー5人以上 |

### 12-5. パートナー制度の設計方針

| 項目 | 方針 | 理由 |
|------|------|------|
| **対象** | 個人コンサルタント・独立系コンサル（招待制） | 品質管理。誰でも参加OKにしない |
| **階層** | 1階層のみ（パートナー→顧客で完結） | MLMと明確に区別 |
| **卸値** | 定価の70%（パートナー取り分30%+自由上乗せ） | SaaS代理店の標準的な水準 |
| **価格設定** | パートナーが自由に設定可 | 付加価値（導入支援等）を上乗せするため |
| **ブランド表記** | 「Powered by HARMONIC insight」の表記を推奨（必須ではない） | パートナーの看板を優先 |
| **サポート** | 技術サポートはHARMONIC insightが対応 | パートナーの負担軽減 |
| **契約形態** | パートナー契約（業務委託ではない） | コンプライアンス上のリスク回避 |
