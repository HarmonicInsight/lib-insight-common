## 1. プレミアムブランド・フィロソフィー

### 1.1 ブランドコンセプト

> **「30年のIT/業務コンサルティングの現場から生まれた、唯一の業務改善AIツール」**

HARMONIC insight は、汎用AIツールではない。
30年にわたるIT/業務コンサルティングの現場で直面してきた課題——
「ヒアリングに時間がかかりすぎる」「分析レポートが属人化する」「提案書作成が深夜作業になる」
——これらを根本から解決するために設計された、**プロフェッショナル専用の業務改善AIツール群**である。

### 1.2 プレミアムブランド原則

HARMONIC insight は以下のプレミアムブランド原則に従う：

| 原則 | 意味 | HARMONIC insight での適用 |
|------|------|--------------------------|
| **品質無妥協** | 機能・体験のすべてにおいて最高水準を追求 | AI Tool Use（6ツール）による本質的な業務自動化。見せかけのAIではなく、実務で使える精度 |
| **専門性の深さ** | 万人向けではなく、特定領域のプロのために設計 | 業務改善・コンサルティングの実務フローに完全特化した設計思想 |
| **体験の一貫性** | あらゆる接点でブランド体験が統一されている | Ivory & Gold デザインシステム、3人格AIコンシェルジュ、統一されたUX |
| **希少性と排他性** | 誰でも使えるものではなく、選ばれたプロのためのツール | FREE版の大幅制限、プロフェッショナル向け機能設計、パートナー経由の導入支援 |
| **ストーリーの力** | 製品の背景にある物語が価値を生む | 「30年の現場経験から生まれた」という唯一無二のオリジン・ストーリー |

### 1.3 価値軸の転換

| | 旧（コスパ訴求） | 新（プレミアム訴求） |
|---|---|---|
| **価値の定義** | 「安くて十分使える」 | 「高くても、これでなければならない」 |
| **比較対象** | Copilot（＝より安い代替品） | 外部コンサルタントの人件費（＝はるかに安い専門ツール） |
| **購買動機** | コスト削減 | 業務品質の向上・時間の創出 |
| **顧客心理** | 「お得だから試す」 | 「プロとして必要だから導入する」 |
| **競合との戦い方** | 価格で勝つ | 専門性と体験で勝つ（価格競争に参加しない） |

## 2. 価格の全面改定（プレミアム価格設計）

### 2.1 価格改定の根拠：コンサルタント時間単価からの逆算

プレミアム価格は「ツールの原価」ではなく「顧客が得る価値」から設計する。

**コンサルタントの時間単価（市場相場）：**
- ジュニアコンサルタント: ¥15,000〜¥25,000/時間
- シニアコンサルタント: ¥30,000〜¥50,000/時間
- マネージャー以上: ¥50,000〜¥100,000/時間

**HARMONIC insight が代替する工数（年間）：**

| 業務 | 従来工数（年間） | ツール導入後 | 削減時間 | 削減価値（時給¥30,000換算） |
|------|----------------|-------------|---------|--------------------------|
| Excelレポート作成・版管理 | 200時間 | 50時間 | 150時間 | **¥4,500,000** |
| ヒアリング実施・議事録作成 | 300時間 | 80時間 | 220時間 | **¥6,600,000** |
| 提案書・スライド作成 | 250時間 | 100時間 | 150時間 | **¥4,500,000** |
| 研修・教育動画制作 | 100時間 | 20時間 | 80時間 | **¥2,400,000** |

→ **年間¥18,000,000相当の価値を創出するツールに、¥98,000〜¥198,000は投資対効果として極めて合理的。**

### 2.4 価格ポリシー（絶対原則）

1. **値下げは一切行わない。** セール、クーポン、期間限定割引は実施しない
2. **価格交渉には応じない。** ENTERPRISE CUSTOMのみボリュームディスカウントを検討
3. **ローンチ価格の廃止。** 「正規価格の40-50%OFF」等のローンチ割引は行わない
4. **年額一括払いのみ。** 月額払いオプションはInterviewInsight（SaaS）のみ
5. **返金保証は品質保証として提供。** 30日間の返金保証は「品質への自信」の表現であり、リスク軽減策ではない

### 3.2 新ペルソナ

#### ペルソナA：コンサルティングファームのマネージャー（主要ターゲット）

- **プロフィール**: 35〜45歳、Big4系または独立系コンサルファームのマネージャー
- **課題**: ヒアリング→分析→提案書のサイクルが属人的で、ジュニアに任せると品質が落ちる
- **予算権限**: ¥500,000/月まで自身の裁量で決済可能
- **情報収集**: LinkedIn、業界カンファレンス、同業者の口コミ
- **購買基準**: 「自分のチームの生産性が上がるか」「品質を維持できるか」
- **刺さるメッセージ**: 「あなたの30年分のノウハウをAIが再現する——のではなく、あなたの判断を加速する」

#### ペルソナB：上場企業の経営企画部長

- **プロフィール**: 45〜55歳、従業員500名以上の企業の経営企画部門責任者
- **課題**: 中期経営計画の策定に外部コンサルを使うと¥50,000,000。内製化したいが人材がいない
- **予算権限**: ¥2,000,000/年まで部門予算で執行可能
- **情報収集**: 日経、東洋経済、業界セミナー、コンサル出身者のネットワーク
- **購買基準**: 「導入実績」「セキュリティ（オフライン対応）」「サポート体制」
- **刺さるメッセージ**: 「外部コンサルの1/50の投資で、コンサルグレードの分析基盤を内製化」

#### ペルソナC：DX推進室のリーダー

- **プロフィール**: 30〜40歳、DX推進を任された中堅社員
- **課題**: 「AIを導入しろ」と言われたが、Copilotは高すぎる（¥4,500/人/月×全社員）。かといって無料ツールでは成果が出ない
- **予算権限**: ¥1,000,000/年のDX予算
- **情報収集**: Qiita、note、X（Twitter）、DX関連ウェビナー
- **購買基準**: 「現場が実際に使えるか」「ROIを上に説明できるか」
- **刺さるメッセージ**: 「Copilot全社導入の1/10の予算で、10倍の業務改善効果」

## 4. 「30年のIT/業務コンサルティング経験」× プレミアムブランドのナラティブ

### 4.1 オリジン・ストーリー

> **30年間、IT/業務コンサルタントとして、数百の企業の現場に立ってきた。**
>
> 航空会社のFFPバックエンド、財務会計パッケージ、コネクテッドカー・プラットフォーム、RPAソリューション——
> 事業運用を止められないミッションクリティカルな領域で、基幹システムを"止めずに進化させる"開発を積み重ねてきた。
>
> その中で何度も同じ光景を見た。
> 優秀なコンサルタントが、深夜までExcelと格闘している。
> 3時間かけたヒアリングの議事録を、さらに3時間かけてまとめている。
> 完璧な分析結果を、PowerPointに転記するだけで半日が消える。
>
> 「この作業を、なぜまだ人間がやっているのか？」
>
> その問いへの答えが、HARMONIC insight だ。
>
> 汎用AIツールは、コンサルティングの「本当の現場」を知らない。
> だから、知っている人間が作った。
> 30年分の「こうあるべき」を、すべて詰め込んだ。

### 4.2 権威構築の段階的アプローチ

**現段階では「30年のIT/業務コンサルティング経験を持つ開発者」を前面に出す。**
具体的な社名・役職は、市場での認知が確立した後に段階的に開示する。

#### フェーズ1（2026年3月〜6月）：経験と実績で語る

- **前面に出すもの**:
  - 「IT/業務コンサルティング30年」の経験値
  - 航空／会計／モビリティ／業務自動化という業界横断のミッションクリティカル領域の実績
  - 「数百社の現場で見てきた課題」という具体性
  - 「コンサルタント自身が、自分のために作ったツール」というオーセンティシティ
- **控えめにするもの**:
  - 具体的な社名・役職
  - 会社の規模や知名度に依存した権威付け
- **チャネル**:
  - LinkedIn: 「業務改善×AI」に関する知見の発信（週1〜2回）
  - note/Qiita: 業務改善の具体的な方法論の記事（月2本）
  - ウェビナー: 「業務改善AIの最前線」（月1回、少人数制）

#### フェーズ2（2026年7月〜12月）：実績が語り始める

- 導入企業の事例が蓄積され始める
- 「〇〇社が導入」という社会的証明が使える段階
- 開発者の経歴をより具体的に開示し始める

#### フェーズ3（2027年〜）：思想的リーダーシップの確立

- 書籍出版:「業務改善AIの設計思想」（仮題）
- 業界カンファレンスでの登壇
- メディア露出の本格化

### 4.3 「二足のわらじ」の扱い

**現段階のメッセージング:**

- 「IT/業務コンサルタントとして30年の経験を持つ開発者が、現場の課題を解決するために作った」
- 具体的な企業名・役職は出さず、**経験の深さと現場感**で信頼を構築する
- 「コンサルタントでもあり、開発者でもある」という二面性は**強み**として表現する
  - 「現場を知っているから、本当に使えるツールが作れる」
  - 「自分自身がユーザーだから、妥協のない設計ができる」

## 5. ブランドコミュニケーションの全面改訂

### 5.1 排除すべき表現（ブラックリスト）

以下の表現は、すべてのマーケティング素材・Web・SNS・PRから完全に排除する：

| カテゴリ | 排除すべき表現 | 理由 |
|---------|--------------|------|
| **価格比較** | 「Copilot比XX%安い」「最大94%安い」 | 安さを売りにするブランドではない |
| **コスパ訴求** | 「コスパ最強」「お得」「お手頃」「手が届く価格」 | プレミアムブランドはコスパで勝負しない |
| **割引表現** | 「今だけXX%OFF」「ローンチ記念価格」「早期割引」 | 値下げはブランド価値を毀損する |
| **低価格強調** | 「安い」「月額たったの〜」「ワンコイン」 | 安さの強調はプレミアム感を消す |
| **汎用表現** | 「誰でも簡単に」「初心者でも」「とりあえず無料で」 | 専門性・プロフェッショナル性と矛盾 |
| **外注比較（過度）** | 「外注30万円→AI 2,700円」 | 99%OFFはバーゲンのイメージ。価値で語る |

### 5.2 採用すべき表現（ブランドボキャブラリー）

| カテゴリ | 推奨表現 | 使用例 |
|---------|---------|--------|
| **品質** | 「コンサルグレード」「プロフェッショナル仕様」「妥協なき品質」 | 「コンサルグレードのAI分析を、あなたのデスクトップに」 |
| **専門性** | 「業務改善に特化」「現場発」「30年の知見を凝縮」 | 「30年のIT/業務コンサルティングの知見を凝縮した、唯一の業務改善AIツール」 |

| **成果** | 「生産性を根本から変える」「本質的な業務改善」「戦略的な時間創出」 | 「提案書作成を自動化し、戦略的思考に集中する時間を創出」 |

| **note / Qiita** | 実践的・具体的。業務改善の方法論を深掘り | 安易な「AIで解決」論、ツール比較記事 |
| **プレスリリース** | 事実ベース、業界インパクトを強調 | 価格の安さを見出しにする、大袈裟な表現 |
| **営業資料** | ROI中心、導入事例ベース、数値で語る | 値引き提案、「他社より安い」比較表 |
| **SNS（X等）** | 知見の共有、業界トレンドへのコメント | フレンドリーすぎる口調、絵文字の多用 |

## 6. 3人格AIのプレミアム・コンシェルジュ化

### 6.1 コンセプトの転換

| | 旧コンセプト | 新コンセプト |
