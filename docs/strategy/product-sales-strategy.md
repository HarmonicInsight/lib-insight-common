## 1. 全製品 価格体系

### 1-1. 価格設計の原則

| 原則 | 理由 |
|------|------|
| **スポット＋サブスクのハイブリッド** | 初期売上（キャッシュ）とMRR（安定収益）の両立 |
| **1クライアントからの年間売上を最大化** | 少人数組織ではクライアント数に限りがある。1社あたりの深さで勝負 |
| **パートナーが売りたくなる利益率** | 卸値70%（パートナー粗利30%＋上乗せ自由）を全製品で統一 |
| **無料枠はフロントドアのみ** | IMA無料診断とInsightCast Freeのみ。他は有料 |

## 4. 販売プロセス（パイプライン）

### 4-1. 共通パイプラインステージ

```
Stage 1: リード獲得
  ├─ FPT紹介    → 即Stage 3（信頼関係あり）
  ├─ パートナー紹介 → Stage 2 or 3
  ├─ Web問い合わせ  → Stage 2
  ├─ PR/SNS流入    → Stage 2
  └─ Free登録      → Stage 2（教育メール自動配信）

Stage 2: 教育・ナーチャリング
  ├─ 教育メール自動配信（InsightCast: 30日シーケンス）
  ├─ YouTube動画視聴誘導
  ├─ ユースケース資料送付
  └─ 期間: 1〜4週間

Stage 3: デモ・トライアル
  ├─ オンラインデモ（30分）
  ├─ トライアル版提供（14日間）
  ├─ PoC提案（InsightMigrationの場合）
  └─ 期間: 1〜2週間

Stage 4: 提案・見積
  ├─ 要件ヒアリング
  ├─ 見積提出（製品＋導入支援）
  ├─ 社内稟議支援資料の提供
  └─ 期間: 1〜4週間

Stage 5: 契約・導入
  ├─ 契約締結
  ├─ 初期設定・テーブル設計（InsightSheet）
  ├─ トレーニング
  └─ 期間: 1〜4週間

Stage 6: 運用・クロスセル
  ├─ 月次利用状況レビュー
  ├─ 追加製品の提案
  ├─ パートナーによる継続支援
  └─ 継続
```
