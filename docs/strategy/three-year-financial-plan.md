## 2. 前提条件

### 2-1. 季節性（会計年度ベース）

会計年度がQ4（4-6月）に日本の新年度を含むため、**季節性が戦略的に有利**:

| 四半期 | 月 | 市場の動き | 自社アクション |
|--------|-----|----------|-------------|
| **Q1** (7-9月) | 夏 | 新年度体制が落ち着く。トライアル→本契約の転換期 | 転換促進・オンボーディング強化 |
| **Q2** (10-12月) | 秋〜年末 | 年度内予算の消化。「年内に導入しよう」 | クロージング集中。年末キャンペーン |
| **Q3** (1-3月) | 冬〜年度末 | 次年度予算の策定。新規ツール選定 | パイプライン構築。デモ集中 |
| **Q4** (4-6月) | 春 | **新年度開始。最大の商機** | PR・新規獲得集中。新入社員研修需要 |

```
季節性と売上のイメージ:

  売上 ┤
       │              ┌──┐
  高   │        ┌──┐  │Q4│  ← 新年度＝最大の商機
       │  ┌──┐  │Q2│  │  │
  中   │  │Q1│  │  │  │  │
       │  │  │  │  │  │  │
  低   │  │  │  │  │  │  │
       │  │Q3│  │  │  │  │  ← 予算策定期（種まき）
       ┼──┴──┴──┴──┴──┴──┴──
        Q1    Q2    Q3    Q4
        7-9  10-12  1-3   4-6
```
