# Insight Series UI システム

## 概要

Insight Series 全製品で統一されたUIを実現するための共通ガイドラインです。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| [menu-structure.json](./menu-structure.json) | メニュー・ナビゲーション構造 |
| [components.md](./components.md) | UIコンポーネント設計ガイド |

## デザインの原則

### 1. 一貫性 (Consistency)

全製品で同じUIパターンを使用し、ユーザーが迷わない体験を提供

### 2. 効率性 (Efficiency)

最小限のステップで目的を達成できるUI設計

### 3. 洗練 (Sophistication)

プロフェッショナル向けの高品質なビジュアル

### 4. アクセシビリティ (Accessibility)

すべてのユーザーが利用できるUI

## 標準レイアウト

```
┌──────────────────────────────────────────────┐
│  Header                                      │
├─────────┬────────────────────────────────────┤
│         │                                    │
│ Sidebar │            Content                 │
│         │                                    │
└─────────┴────────────────────────────────────┘
```

## 共通メニュー項目

すべての製品に含まれる標準メニュー:

- ホーム
- ダッシュボード
- プロジェクト
- 最近使用
- 設定
- ヘルプ

## キーボードショートカット

| ショートカット | 機能 |
|--------------|------|
| Ctrl+N | 新規作成 |
| Ctrl+S | 保存 |
| Ctrl+K | 検索 |
| Ctrl+, | 設定 |
| F1 | ヘルプ |
