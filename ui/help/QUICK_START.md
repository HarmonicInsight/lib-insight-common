# InsightOfficeSheet クイックスタートガイド

> InsightOfficeSheet は、Excel ファイルの読み書き・数式解析・マクロ実行・一括処理ができるツールです。
> ITの専門知識がなくても、かんたんに使えるように設計されています。

---

## はじめに

### InsightOfficeSheet でできること

| 機能 | 説明 | プラン |
|------|------|--------|
| Excel読取 | Excel ファイルの中身を読み取る | 全プラン |
| Excel書込 | Excel ファイルにデータを書き込む | 全プラン |
| 数式解析 | セルの数式を解析・変換する | STD以上 |
| マクロ実行 | VBA マクロを実行・変換する | PRO以上 |
| バッチ処理 | 複数ファイルをまとめて処理する | PRO以上 |
| テンプレート | テンプレートから Excel を生成する | PRO以上 |

---

## Step 1: アプリを起動する

1. デスクトップの **InsightOfficeSheet** アイコンをダブルクリック
2. 初回起動時はライセンス認証画面が表示されます

### ライセンス認証

```
┌────────────────────────────────────┐
│       InsightOfficeSheet                │
│                                    │
│  メールアドレス: [          ]      │
│  ライセンスキー: [          ]      │
│                                    │
│      [アクティベート]              │
└────────────────────────────────────┘
```

- 購入時に届いたメールに記載されているライセンスキーを入力してください
- ライセンスキーの形式: `IOSH-STD-2601-XXXX-XXXX-XXXX`
- トライアル版は1ヶ月間すべての機能が使えます

---

## Step 2: Excel ファイルを開く

### 方法1: メニューから開く

1. 左側メニューの「**Excelファイルを開く**」をクリック
2. ファイル選択ダイアログが表示されます
3. 対象の Excel ファイル（.xlsx / .xls）を選択
4. 「開く」をクリック

### 方法2: ドラッグ＆ドロップ

- Excel ファイルをアプリのウィンドウに直接ドラッグ＆ドロップ

### 方法3: ショートカットキー

- **Ctrl + O** で「ファイルを開く」ダイアログを表示

## Step 4: データを編集・書き込む

### セルの値を変更する

1. 変更したいセルをクリック
2. 新しい値を入力
3. Enter キーで確定

### 書き込みを保存する

- **Ctrl + S** で上書き保存
- **Ctrl + Shift + S** で「名前を付けて保存」

> **注意**: 保存すると、元のファイルと同じフォルダに履歴ファイル（.hsheet）が自動作成されます。
> 詳しくは「[履歴ファイルについて](./HISTORY_FILES.md)」をご覧ください。

---

## Step 5: 便利な機能を使う

### 数式解析（STD プラン以上）

1. メニューから「**数式解析**」を選択
2. 解析対象のシートとセル範囲を指定
3. 数式の依存関係や計算結果を確認できます

### バッチ処理（PRO プラン以上）

1. メニューから「**バッチ処理**」を選択
2. フォルダを指定して複数ファイルをまとめて処理
3. 処理結果はレポートとして出力されます
